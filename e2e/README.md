# Kvell App Testing

## Get started

### Installation of detox cli

```sh
npm install -g detox-cli
```

### Install apple utils and wix

```sh
brew update && brew install node
```

```sh
brew tap wix/brew;
brew install applesimutils;
```

### Usage

#### Build

```sh
npm e2e:build

```

#### Run all tests

```sh
npm e2e:test

```
